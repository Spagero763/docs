# Build an App on Base - 2026 Guide

Welcome to the updated Base quickstart guide! This walkthrough will help you create a modern onchain application using the latest tools and best practices for 2026.

## What You'll Build

By the end of this guide, you'll have created a production-ready onchain application featuring:

- **Modern Development Stack**: Next.js 15 with App Router and OnchainKit v2
- **Smart Contract Deployment**: Deploy to Base Sepolia (testnet) and Base Mainnet
- **Gasless Transactions**: Implement sponsored transactions for better UX
- **Onchain Identity**: Integrate Base Names and avatars
